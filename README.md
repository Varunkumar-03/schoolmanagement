# 🎓 Happy Tuitions - School Management System

A modern, responsive, and user-friendly School Management System developed for **Happy Tuitions** to simplify the day-to-day administrative tasks of educational institutions.

The application enables administrators to manage students, faculty, fees, events, attendance, and payment tracking from a single dashboard.


## 📌 Project Overview

Happy Tuitions School Management System is a web-based application built to automate institute management processes, reduce paperwork, and improve operational efficiency.

The system provides an intuitive interface for administrators to manage student records, faculty information, fee collection, events, and reports in real time.
